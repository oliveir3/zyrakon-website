# Zyrakon Corporate Website

**The Foundation of Intelligent Technology.**

This repository contains the corporate website for Zyrakon, a technology holding company building intelligent, secure, and trusted platforms for enterprises worldwide.

## Structure

```
zyrakon/
├── pages/          # Individual page HTML files
├── assets/         # CSS, JS, images, fonts, documents
├── components/     # Reusable HTML components
├── data/           # Structured data files
├── scripts/        # Build and automation scripts
├── .github/        # GitHub Actions and templates
└── docs/           # Project documentation
```

## Technology Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Static Generation:** Custom build scripts
- **Deployment:** GitHub Pages / Cloudflare Pages
- **Design System:** Custom Zyrakon Design Language

## Getting Started

```bash
# Clone the repository
git clone https://github.com/oliveir3/zyrakon-website.git

# Open in browser
start index.html
```
